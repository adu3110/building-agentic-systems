# 23. Introduction

Book 1: one agent that acts.  
Book 2: measure whether it acts correctly.  
Book 3: **multiple agents** that coordinate without chaos.

CaseBot grows into a team:

- **Investigator** — gathers account data
- **Policy** — checks constraints
- **Resolver** — adjudicates conflicts
- **Auditor** — replays ledger for compliance

They do not message each other in a group chat. They read and write a **shared append-only ledger**.
